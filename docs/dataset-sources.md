## Meta catalogs / search
- Data.gov (US government open data portal) advertises **386,538 datasets available** and is a good starting point for public-sector dashboards. Source: https://data.gov/
- Google Dataset Search is a meta search engine for datasets. Source: https://datasetsearch.research.google.com/

## City / municipal open data
- NYC Open Data provides a browsable catalog and APIs for city datasets (crime, 311, permits, transportation, etc.). Source: https://opendata.cityofnewyork.us/
