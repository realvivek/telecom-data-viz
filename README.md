Track NTIA BEAD broadband funding, last-mile coverage gaps, and fiber deployment opportunities across US counties.

## Features
- County-level data: 30 major US counties
- Filter by state, priority (High/Med/Low), and minimum score
- Charts: funding by state, gap distribution, scatter plot, priority donut
- Insights tab: curated opportunities and analysis
- Dark mode, responsive

## Data Sources
- FCC BroadbandMap
- NTIA BEAD Program grants database
- Census ACS 5-year estimates

## Stack
HTML + Chart.js (CDN). Zero backend required.
